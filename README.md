# lndpnr Theme

![lndpnr theme preview](https://github.com/user-attachments/assets/368e13a5-43b3-4062-9d0b-37f3b61af67e)

A dark theme for Zed editor by Leandro Pinheiro Rosa.

## Installation

1. Open Zed editor.
2. Go to `Extensions`.
3. Search for `lndpnr` and install it.

## Contributing

If you'd like to contribute to this theme, feel free to open an issue or pull request on the [GitHub repository](https://github.com/leaandropinheiro/lndpnr).
